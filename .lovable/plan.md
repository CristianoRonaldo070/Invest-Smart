

# InvestSmart - Investment & Savings Platform (Frontend)

## Design Theme
- **Dark background** with deep navy/charcoal tones
- **Blue accent colors** (electric blue gradients, glowing effects)
- Smooth animations, glassmorphism cards, modern typography

## Pages & Features

### 1. Splash/Loading Page
- Animated investment-themed intro (rising chart animation, coin/growth icons)
- Auto-redirects to login after 3 seconds
- Glowing blue animated elements

### 2. Login & Sign Up Pages
- Clean dark-themed forms with blue accents
- Toggle between Login and Sign Up
- Stores user info in localStorage (no backend)
- Form validation with error messages

### 3. Home Page - Investment Hub
- **Hero section** with investment tips carousel
- **Investment categories grid**: Stocks, Mutual Funds, Fixed Deposits, Gold, Real Estate, Crypto
- Each category card shows brief details, risk level, and expected returns
- "Investment Tricks & Tips" section with actionable advice cards
- Navigation bar with links to all sections

### 4. Savings & Expenses Tracker
- Add income and expenses with categories
- Visual breakdown (progress bars/charts showing savings vs expenses)
- When savings are detected, **AI-powered investment suggestions** appear automatically
- Suggestions include where to invest, why, risk level, and expected returns
- Data stored in localStorage

### 5. AI Chatbot
- Floating chat button on all pages
- Chat interface where users can ask about saving money, investment strategies, budgeting tips
- Powered by Lovable AI for real, intelligent responses
- Markdown-rendered responses with streaming

### 6. Layout & Navigation
- Persistent dark sidebar or top navbar
- Smooth page transitions
- Mobile responsive design
